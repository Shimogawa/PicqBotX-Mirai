### 注意事项

将不会实现以下内容

- 原有的 HTTP API
- 多账号

原有的某些 API 将会保留，但会标注`@Deprecated`并且添加新 API，因为名称已经不适用于基于 Mirai 的新版本（例如`IcqListener`将会变成`EventListener`，但`IcqListener`将会保留）。
